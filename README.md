## users table

|Colum|Type|Option|
|-----|----|------|
|name|string|index: true, null: false, unique: true|
|email|string|null: false, unique: true|
|password|string|unique:true|

### Association

-has_many :groups, through::group_users table

-has-many :group_users

-has_many :messages
